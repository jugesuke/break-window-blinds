# ブラインド崩し

未来大のブラインドを開けるブロック崩しゲーム

## Features

- 時間が経つにつれて背景と画面の明るさが変わる
- 仕事を助けてくれるアイテムが使えるが、学内各店舗の営業時間内でしか購入できない
- コンビニで売っているアイテムは購入に時間がかかり、その間ゲームが停止する

## Requirements

- Processing 3.5.4
